# Hack-the-Box-SOC-Analyst-Path
A documentation of my progress through the Hack The Box SOC Analyst learning path. This repository contains notes, methodology breakdowns, and key takeaways from each module — written to demonstrate practical blue team skills to recruiters and hiring managers.



About This Repository
The HTB SOC Analyst Path is a structured curriculum designed to build real-world skills used by Security Operations Center analysts. Rather than just listing completions, this repository captures how I approached each domain, what tools I used, and what I learned along the way.
Skill areas covered:

SIEM analysis and query writing
Windows Event Log forensics
Network traffic analysis
Phishing investigation
Malware traffic analysis
Threat hunting techniques
Incident triage and response workflows
